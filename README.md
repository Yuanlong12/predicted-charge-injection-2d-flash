# Multi-bit programming of Ultrafast 2D flash memory with Predicted Charge Injection

This repository contains the Python simulation code associated with the manuscript:

**"Multi-bit programming of Ultrafast 2D flash memory with Predicted Charge Injection"**

## Description

This repository provides Python code for simulating charge injection, floating-gate potential evolution, and threshold-voltage shift in ultrafast 2D flash memory devices.

## Files

- `FGT_model_ver0.py`: core simulation model
- `predictive_programming_main.ipynb`: main notebook for simulation and analysis

## Requirements

The code is based on Python and requires common scientific computing packages, including:

- numpy
- scipy
- matplotlib
- jupyter

## Usage

You can run the Python script directly or open the notebook for simulation and visualization.
